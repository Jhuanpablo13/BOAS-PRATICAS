# Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente

# Boas práticas

📋 Sobre o Projeto
O projeto sobre boas práticas é uma página web interativa desenvolvida para incentivar e compartilhar práticas sustentáveis no agronegócio. O projeto foi criado pelo jhuan Pablo do Colégio Estadual Rio Branco com o objetivo de promover o equilíbrio entre produção agrícola e preservação ambiental.

A aplicação permite que usuários compartilhem fotos e vídeos de iniciativas sustentáveis, criando uma comunidade colaborativa de conhecimento e boas práticas.

✨ Funcionalidades
📤 Compartilhamento de conteúdo – Envie fotos e vídeos de práticas sustentáveis

🏷️ Sistema de categorias – Agricultura sustentável, reciclagem, energia renovável, preservação, produção orgânica, compostagem e mais

🔍 Filtros inteligentes – Visualize apenas fotos, apenas vídeos ou todo o conteúdo

📊 Estatísticas em tempo real – Acompanhe o total de contribuições, fotos e vídeos

🏷️ Tags personalizáveis – Adicione tags para facilitar a busca e organização

🖼️ Pré-visualização – Visualize o arquivo antes de publicar

⌨️ Comandos por terminal – Interface interativa com comandos de ajuda

📱 Design responsivo – Funciona em dispositivos móveis e desktop

🛠️ Tecnologias Utilizadas
HTML5 – Estrutura semântica da aplicação

CSS3 – Estilização moderna com grid, flexbox e animações

JavaScript (Vanilla) – Lógica da aplicação, manipulação do DOM e localStorage

Font Awesome – Ícones vetoriais para melhor experiência visual

LocalStorage – Persistência de dados no navegador

🎯 Objetivos do Projeto
Incentivar práticas agrícolas sustentáveis

Documentar e compartilhar experiências reais

Criar uma comunidade de aprendizado colaborativo

Conscientizar sobre a importância do equilíbrio entre produção e meio ambiente

💻 Como Usar
Compartilhar conteúdo

Clique no botão "Compartilhar Foto ou Vídeo"

Preencha seu nome, título, categoria e descrição

Selecione o arquivo de mídia (imagem ou vídeo)

Adicione tags (opcional)

Clique em "Publicar"

Filtrar publicações

Use os botões "Todos", "Fotos" ou "Vídeos" para filtrar o conteúdo

Comandos especiais

Digite /ajuda no terminal interativo para ver todos os comandos disponíveis
